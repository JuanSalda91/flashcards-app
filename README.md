# Getting Started

<h2>Flashs Cards App</h2>
<p>This application is the perfect to to help you study and help you remember the main ideas of any topic you want to lern more about</p>

1. Open the app: Open index.html in your browser
2. Create a deck: Click "New Deck" in the sidebar, enter a name (e.g., "Spanish 101")
3. Add cards: Click "New Card" in the main area, enter front (question) and back (answer) text

## Studying

1. Enter study mode: Click on a deck name in the sidebar
2. Flip cards: Press Space or click the flip button to see answers
3. Navigate: Use ‚Üê ‚Üí arrow keys or click Previous/Next buttons
4. Exit: Press Escape or click "Exit Study Mode"

## Managing Cards

- Edit: Click the pencil icon on a card in the card list
- Delete: Click the trash icon (with confirmation)
- Shuffle: Click "Shuffle" to randomize card order

## Searching

- Find cards: Type in the search box to filter cards by front/back text
- View matches: See count of matching cards in the top right
- Clear search: Empty the search box to restore all cards

## Data Persistence

- Auto-save: All changes (decks, cards, edits) save automatically to browser storage
- Survives reload: Close the browser and your decks will be there when you return
- No account needed: Data stored locally on your device
